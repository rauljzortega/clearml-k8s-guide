# ClearML Deployment in Kubernetes  

This repository contains a guide for the installation and deployment of ClearML on Kubernetes, as well as the bootstrapping of the Kubernetes cluster itself.  

**If you already have a running Kubernetes cluster**, go directly to the [ClearML installation guide](./clearml/clearml-k8s-installation.md).  

**If you need to set up your cluster from scratch**, follow the [Kubernetes installation guide using kubeadm](./kubernetes/kubernetes-installation.md) first.
